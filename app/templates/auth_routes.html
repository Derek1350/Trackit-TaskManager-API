Trackit Auth Reference Guide

Note: You must register first to create a user. Then, login to receive a JWT token which is valid for 1 hour. After that, you need to login again to get a new token.

üìå Register a New User

POST /auth/register

Request Body (JSON)

{
    "username": "johndoe",
    "email": "john@example.com",
    "password": "securepassword",
    "role": "user"  // optional, default = "user"
}

Validations

  • username and password are required
  • email must be unique

Response (201)

{ "message": "User registered successfully" }

üîê Login

POST /auth/login

Request Body (JSON)

{
    "username": "johndoe",
    "password": "securepassword"
}

Response (200)

{
    "access_token": ""
}

Failure (401)

{ "message": "Invalid credentials" }
