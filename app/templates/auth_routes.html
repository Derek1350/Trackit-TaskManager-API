TaskPulse API Reference Guide

üìå Register a New User

POST /auth/register

Request Body (JSON)

{
    "username": "johndoe",
    "email": "john@example.com",
    "password": "securepassword",
    "role": "user"  // optional, default = "user"
}

Validations

  • username and password are required
  • email must be unique

Response (201)

{ "message": "User registered successfully" }

üîê Login

POST /auth/login

Request Body (JSON)

{
    "username": "johndoe",
    "password": "securepassword"
}

Response (200)

{
    "access_token": ""
}

Failure (401)

{ "message": "Invalid credentials" }

üìÑ Upload CSV to TaskManager

POST /upload-csv

Headers

Authorization: Bearer <JWT-TOKEN>

CSV Format

title,description,priority
"Build Backend","Complete backend logic","high"
"Write Docs","Document all endpoints","medium"

Response

{ "message": "Tasks uploaded successfully" }

Error

{ "error": "Invalid file or format" }
